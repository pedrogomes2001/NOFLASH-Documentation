# Especificação da API


## Users

**Endpoint - HTTP Action - Business Operation**

/Users - GET - getAllUsers

/Users/{user_id} - GET - getUserId

/Users/{user_id} - PUT - updateUser

/Users/{user_id} - DELETE - deleteUser
